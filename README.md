# UnitTestInVue

Vue Test Utils（VTU）是一组实用程序功能，旨在简化对Vue.js组件的测试。 它提供了一些以隔离方式安装Vue组件并与之交互的方法。视频通过实例介绍了如何通过VTU对Vue组件进行单元测试。主要涉及内容包括：

    ⌨️ Vue组件单元测试基本介绍

    ⌨️ Snapshot测试

    ⌨️ Vuex测试
    
    ⌨️ Vuetify
    
    ⌨️ 针对键盘,鼠标等事件测试
    
作者通过现场查阅资料+编码的方式，让整个介绍更生动有实战性。
